# Amplify Books (AWS Amplify Gen 2 Demo)

A minimal Amplify project with:
- React frontend
- Amplify Gen 2 backend (GraphQL + DynamoDB)

## Deploy
1. Push this repo to GitHub.
2. In the AWS Console → Amplify → “Get started”.
3. Choose “Connect app”, select this repo + branch.
4. Amplify detects the backend and frontend automatically.
5. Click “Deploy”.

After deploy, open Amplify Studio → Data → you’ll see the `Book` table.
